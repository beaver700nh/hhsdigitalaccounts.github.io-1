Critical Thinking for Web Research

OBJECTIVE: to learn how to define and ask questions for Web research

Good information can be hard to find, whether you're looking in books, asking friends, or surfing the World Wide Web. But knowing how to find reliable, useful information on the Web is particularly important because so much information is available. Once you know how to ask the right questions, use search engines, and evaluate the results, you can harness the Web to conduct research on almost any topic. In this activity you will practice asking questions that will help you search the Web more efficiently.

Math Review

OBJECTIVE: to review exponents and to practice using exponents to write numbers in scientific notation

In science, it is sometimes necessary to deal with numbers that are so large or so small that they are difficult to write and understand.

For example: the human body contains 100,000,000,000,000 (one hundred trillion) cells.

One way to represent such numbers more understandably is using scientific notation. Scientific notation is also known as exponential notation because it involves using exponents to express a number. An exponent is basically shorthand for multiplication; it indicates how many times the base is to be multiplied by itself.

For example: 53 is 5 raised to the 3rd power: 5 x 5 x 5 = 125.

In scientific notation, the base is always 10. In this activity, you will practice converting numbers to scientific notation, and learn how to manipulate exponents when doing calculations. When you are finished, scroll down to answer the question below.

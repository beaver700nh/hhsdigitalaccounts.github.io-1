OBJECTIVE: to learn how Mendel explained the results of a monohybrid cross

Mendel hypothesized that an organism carries two alleles for each character. In the case of the pure-breeding purple-flowered pea plant, the alleles can be described as PP. For the pure-breeding white-flowered pea plant, the alleles can be described as pp. Mendel stated that each parent contributes only one of the two alleles to each offspring.
